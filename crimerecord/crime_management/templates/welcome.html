About This System

The Crime Record Management System is a comprehensive solution designed to facilitate the efficient management of crime-related data for law enforcement agencies. It provides a centralized platform for recording, tracking, and analyzing crime incidents, criminal profiles, and police operations.

Built using modern technologies including Django, Bootstrap, and Chart.js, the system offers a user-friendly interface with responsive design for optimal usability across devices.

Technical Specifications

  • Backend: Django (Python)
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • Database: MySQL / SQLite
  • Data Visualization: Chart.js
  • Enhanced Tables: DataTables
  • Icons: FontAwesome
