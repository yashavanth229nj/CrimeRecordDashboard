About This System

The Crime Record Management System is a comprehensive solution designed to facilitate the efficient management of crime-related data for law enforcement agencies. It provides a centralized platform for recording, tracking, and analyzing crime incidents, criminal profiles, and police operations.

Built using modern technologies including Django, Bootstrap, and Chart.js, the system offers a user-friendly interface with responsive design for optimal usability across devices.

Technical Specifications

  • Backend: Django (Python)
  • Frontend: HTML, CSS, JavaScript, Bootstrap 5
  • Database: MySQL
  • Data Visualization: Chart.js
  • Enhanced Tables: DataTables
  • Icons: FontAwesome

Local Implementation Guide

To get the Crime Record Management System working on your local machine with the MySQL database:

  1. Ensure MySQL is running locally with the credentials specified in the settings.py file.
  2. Create the database using the SQL script provided in the attached file.
  3. Install required dependencies: 
    pip install django mysqlclient
  4. Run the development server locally: 
    cd crimerecord
python manage.py runserver
  5. Access the application at http://localhost:8000 in your web browser.
Note:

This system is configured to connect to a MySQL database running locally on your machine with the following settings:

  • Database: CrimeRecord
  • User: root
